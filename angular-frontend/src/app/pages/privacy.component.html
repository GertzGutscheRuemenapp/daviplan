  2. Datenverarbeitung im Zusammenhang mit der Bereitstellung des Tools „daviplan“

    3. „daviplan“ ist ein Datentool, welches im Rahmen des Förderprogramms „Region gestalten“ von unserem Auftragnehmer Gertz Gutsche Rümenapp – Stadtentwicklung und Mobilität GbR (GGR) entwickelt und realisiert worden ist.

    Dieses Tool wird als Docker-Image auf der Webseite https://hub.docker.com zum Download bereitgestellt. Es ist zum eigenem Hosting und zur eigenen Verwendung durch die Regionen / Kommunen zum Zwecke der regionalen Daseinsvorsorge konzipiert.

    Für Interessierte wird der Code zusätzlich auf der Webseite https://github.com/ zum Download bereitgestellt.

    Die Datenverarbeitung im Tool „daviplan“ findet ausschließlich bei den Regionen / Kommunen selbst statt. Weder das BBR noch das Büro GGR haben Zugriff auf die von den Verwendenden eingegebenen (personenbezogenen) Daten. Es findet somit keine Verarbeitung personenbezogener Daten seitens des BBR oder des Büros GGR in diesem Tool statt.

    Die Webseite https://hub.docker.com dient alleinig der Bereitstellung des Tools für die lokale Installation. Während der Toolanwendung werden keine Daten an die Webseite hub.docker.com übermittelt. Eine Datenverarbeitung im Zusammenhang mit dem Tool „daviplan“ findet in dem Moment statt, in dem die Region/Kommune die Website https://hub.docker.com zum Zwecke des Downloads des Tools aufruft. Die Website https://hub.docker.com wird von der Docker Inc. betrieben. Docker Inc. ist ein Unternehmen mit Sitz in Kalifornien, Vereinigte Staaten von Amerika (USA). Daher handelt es sich bei Docker Inc. um ein Unternehmen in einem Drittland, die Verarbeitung personenbezogener Daten findet folglich nicht in der Europäischen Union (EU), sondern in einem Drittland statt. Für weitere Informationen zu einer möglichen Verarbeitung personenbezogener Daten beim Aufruf der Website https://hub.docker.com lesen Sie bitte die dortigen Hinweise zum Datenschutz .

    Des Weiteren findet eine Datenverarbeitung im Zusammenhang mit dem Tool „daviplan“ in dem Moment statt, in dem die Region/Kommune die Website https://github.com aufruft. GitHub ist ein Unternehmen der Microsoft Corporation mit Sitz in Kalifornien, Vereinigte Staaten von Amerika (USA). Somit handelt es sich bei GitHub um ein Unternehmen in einem Drittland, die Verarbeitung personenbezogener Daten findet folglich nicht in der Europäischen Union (EU), sondern in einem Drittland statt. GitHub verarbeitet personenbezogene Daten gemäß den Angaben, die in deren Hinweisen zum Datenschutz aufgeführt sind. Das BBR hat hierauf keinerlei Einfluss.

    Das BBR hat auf Datenverarbeitungen der Website-Betreiber leider keinerlei Einfluss.

    Zur lokalen Installation von daviplan in einer Kommune oder Region ist die eigene Bereitstellung oder Anmietung eines Servers durch die nutzende Kommune oder Region erforderlich. Die Anwendung „daviplan“ speichert personenbezogene Daten sodann auf diesem Server. Entsprechend sind die anwendenden Kommune oder Region für die Einhaltung der Datenschutzbestimmungen auf diesem von Ihnen selbst bereitgestellten oder angemieteten Server verantwortlich.
